# pingtocsv
ping to csv

## features
* runs ping once then waits 5 seconds
* captures the date time for each ping
* saves the output to a .csv file

## windows executable
download exe from releases and double click

## build
* clone
* pip install .[test]
* `pyinstaller --onefile pingtocsv/pingtocsv.py`
